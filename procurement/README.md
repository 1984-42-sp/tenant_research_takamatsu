# procurement

高松市でのカフェ開業に向けた仕入れ調査プロジェクト。

## 目的

フルーツ、コーヒー豆、消耗品の仕入先を調査し、価格・季節変動・配送可否・公式URLを整理する。

## 調査ジャンル

1. フルーツ仕入れ
2. コーヒー豆仕入れ
3. 消耗品仕入れ

## 最終的な統合先

完成後、tenant_research_takamatsu/procurement/ に移動し、既存の物件・競合店舗調査ダッシュボードと統合する。

## 主要データ

- data/suppliers_master.csv
- data/fruit/fruit_suppliers.csv
- data/fruit/seasonal_fruit_prices.csv
- data/coffee/coffee_suppliers.csv
- data/consumables/consumables_suppliers.csv
